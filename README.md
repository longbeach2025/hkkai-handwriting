# hkkai-handwriting
hkkai-handwriting generates A4 PDF practice sheets for Hong Kong Traditional Chinese. CSV-driven (part,char,jyut,examples) to “Satisfaction v1.0.4” layouts: titles/footers, Cangjie5 first+last keys, Jyutping, Cantonese examples, 10×15 mm Mi-grid boxes, subtle watermark. Includes robust CSV cleaner and sample assets.
